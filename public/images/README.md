## Instructions:
1. Add your PNG images to this folder
2. Name them as: gym-1.png, gym-2.png, etc.
3. The gallery will automatically display them
4. If images are not found, placeholder images will be shown

## Recommended Image Sizes:
- Width: 800-1200px
- Aspect Ratio: 4:3 or 16:9
- Format: PNG or JPG
- File Size: Under 500KB each for best performance
